# kleaniq-webapp

> Web interface for KleanIQ

## Getting Started

Getting up and running is as easy as 1, 2, 3.


2. Install your dependencies

    ```
    yarn install
    ```

3. Build latest release and serve

    ```
    yarn start
    ```

    start development  & app server

    ```
    yarn build
    ```

    build production

    ```
    yarn serve
    ```

    serve production
